# MakeMyTrip Clone - Angular UI

A modern Angular-based user interface clone of MakeMyTrip, India's leading travel booking platform. This project replicates the core UI components with responsive design and interactive features.

 Features

- **Flight Search** - One-way, round trip, and multi-city search interface
- **Interactive Flight Cards** - Searchable city selection with 100+ destinations
- **Special Fares Section** - Student, Armed Forces, Senior Citizen, Doctor discounts
- **Visa-Free & E-Visa Destinations** - Quick destination chips
- **Zero Cancellation / FlexiFly** - Refund options
- **Responsive Design** - Works on all devices
- **Bootstrap 5** - Modern UI components
- **Font Awesome Icons** - Beautiful icon set

 Technologies Used

- **Angular** 18+
- **Bootstrap 5** - For responsive layout
- **Font Awesome** - Icons
- **TypeScript** - Logic and interactivity
- **HTML5/CSS3** - Structure and styling
